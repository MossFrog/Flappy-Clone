# Flappy-Clone
Simple flappy bird clone in C++ using SFML.

Included are the compiled executable and all the C++ Files.

Controls:
Use the 'G' key to start the game and restart.
'Space' key to make the bird flap.

Future Updates

-Working Menu
-Mortal Objects
-Lightning
